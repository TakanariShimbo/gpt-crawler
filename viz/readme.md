# About

URL Hierarchy Visualizer

# Create Venv

```
conda create -n python310_gpt_crawler_viz python=3.10 -y
conda activate python310_gpt_crawler_viz
```

# Run

```
python ./viz/viz_url_hierarchy.py --input ./res/uniontool.json --output ./res/uniontool.md
```
